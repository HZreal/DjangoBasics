{# 过滤器(本质就是函数)语法： 变量|过滤器:参数 注意英文冒号后不能有空格 #} 我的生日是{{ birthday|date:'Y年 m月 d日' }}
我的简介是{{ desc }} {# 不会执行，直接显示 #}
我的简介是{{ desc|safe }} {# safe告知此变量安全，可以执行。于是模板会执行对应js代码 #}
未定义的变量：{{ catr|default:'变量默认值----' }} {# 设置默认值，若变量不存在则返回默认值 #}
我有{{ friends|length }}个朋友 {# 返回字符串包含字符个数，或列表元组字典的元素个数 #}
